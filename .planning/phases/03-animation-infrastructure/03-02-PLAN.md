    **1. core-glow keyframe (opacity only -- compositor-friendly):**
    ```css
    @keyframes core-glow {

      animation: core-glow 2s ease-in-out infinite;
    }
    ```

    **2. data-flow keyframe (stroke-dashoffset -- compositor-friendly for SVG):**
    ```css
    @keyframes data-flow {
      to { stroke-dashoffset: -20; }
    }
    .animate-data-flow {
      stroke-dasharray: 8 4;
      animation: data-flow 1s linear infinite;
    }
    ```

    **3. memory-shimmer keyframe (transform only -- compositor-friendly):**
    ```css
    @keyframes memory-shimmer {
      0% { transform: translateX(-100%); }
      100% { transform: translateX(100%); }
    }
    .animate-memory-shimmer {
      animation: memory-shimmer 3s ease-in-out infinite;
    }
    ```

    **4. Blanket reduced-motion disable (MUST be last CSS rule so it overrides everything):**
    ```css
    @media (prefers-reduced-motion: reduce) {
      *, *::before, *::after {
        animation-duration: 0.01ms !important;
        animation-iteration-count: 1 !important;
        transition-duration: 0.01ms !important;
      }
    }
    ```

    **Critical constraints (A11Y-02):**
    - core-glow animates ONLY `opacity` (never `fill`, `stroke`, or `filter`)
    - data-flow animates ONLY `stroke-dashoffset` (compositor-friendly for SVG paths)
    - memory-shimmer animates ONLY `transform: translateX` (never `width`, `left`, or `margin`)
    - The reduced-motion rule uses near-zero duration (0.01ms) not duration: 0 to avoid breaking CSS animation events
